# 🌿 Organic Food E-commerce Landing Page

A clean, responsive landing page for an organic food store. This project focuses on modern web layout techniques and mobile-first design to create a pleasant shopping experience.

---

## ✨ Features

* **Responsive Design:** Fully optimized for mobile, tablet, and desktop using CSS Media Queries.
* **Modern UI:** Focused on high-quality imagery and clean typography to reflect the "organic" brand identity.
* **Semantic Structure:** Built with HTML5 semantic elements for better SEO and accessibility.
* **Navigation:** Functional multi-section scrolling layout.

---

## 🛠 Tech Stack

* **HTML5:** Semantic markup for document structure.
* **CSS3:** Custom styles, Flexbox layouts, and responsive design patterns.
* **FontAwesome:** Integrated for intuitive iconography.
* **Google Fonts:** Custom typography for enhanced visual appeal.

---

## 🧩 Development Focus

In this project, my goal was to:
1.  **Master Responsive Layouts:** Ensuring the grid and content scale correctly across different screen resolutions.
2.  **Visual Hierarchy:** Using spacing and font weights to guide the user's eye toward the Call-to-Action (CTA) buttons.
3.  **Cross-Browser Testing:** Ensuring consistent rendering in modern browsers.
